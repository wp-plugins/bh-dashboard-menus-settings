This is BH Dashboard Menus Settings plugin.It will be enable Option to change menus name on your dashboard. 

== Description ==

BH Dashboard Menus Settings plugin.You can change your dashboard menus name.From Settings page you can change your all menus name have a nice day

== Installation ==

1. Upload `bh-dashboard-menus-settings` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
2. After installation go to "BH Dashboard menus settings" settings page.

== Frequently asked questions ==

= How i can use this plugin =

After Installation just go to settings page and you can changes all dashboard menus name.
